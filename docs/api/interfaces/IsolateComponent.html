Interface IsolateComponent

This is the type of the main entry point: isolateComponent()

It accepts a React Element that is a modern react component, usually created with JSX, and returns an IsolatedComponent that provides methods for manipulating and checking the results of rendering that component.

example

Quick start

import { isolateComponent } from 'isolated-components'
const Hello = (props) => <h2>Hello {props.name}</h2>
const component = isolateComponent(<Hello name="Zaphod" />)

console.log(component.toString()) // => "<h2>Hello Zaphod</h2>"

isolateComponent also exposes the method isolateComponent.withContext for setting context values for testing.
